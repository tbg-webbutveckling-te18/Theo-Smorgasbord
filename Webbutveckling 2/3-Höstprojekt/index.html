If/Else-satser

Inom JavaScript, såväl som inom de andra programmeringsspråken finns det Selektioner, exempelvis If-satser.

En If-sats används för att välja eller välja bort saker, alternativ eller händelser, men även för att kolla dess validitet. Det kan exempelvis röra sig om att kolla huruvida ett påstående är sant eller falskt och som följd av det göra något som specificerats i koden. Det skulle kunna vara att starta ett spel när du klickar på ett specifikt element på hemsidan, eller i programmet om vi tänker i andra programmeringsspråk.

Finns kodexempel nedan:

var x = 10

if (x <= 11) {

x += 1;

Console.Writeln(x);

}

For-sats

En For-loop är en looping av argument som repeterar något ett antal gånger. Det kan röra sig om en gång eller 1000 gånger, beroende på vad som specificerats i for-loopen. Exempelvis så kan en For-loop loopa igenom ett antal siffror, den ena efter den andra och skriva ut dem eftersom. 1, 2, 3, 4, 5 osv.

Finns kodexempel nedan:

for(var i = 0; i <= 82; i++) {

document.writeln('Count is ' +i + '
');

}

Array

En Array är en sorts lista som kan innehålla olika föremål, såsom variabler eller objekt. En array har ett index, som numrerar innehållet från 0 och uppåt. Det tredje föremålet i en array ligger på index 2.

Finns kodexempel nedan:

var jobArr = ["President", "Programmer", "Carpenter" ];

Object

Ett Objekt är också en sorts variabel, men denna kan innehålla många värden. Ett objekt har inget index.

Finns kodexempel nedan:

var user = {

name: "Devin",

age: 28,

city: "London",

friends: ["Bob", "Ross"],

isEvil: true

}
