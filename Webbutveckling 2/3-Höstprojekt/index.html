If/Else-satser

Inom JavaScript, såväl som inom de andra programmeringsspråken finns det Selektioner, exempelvis If-satser.

En If-sats används för att välja eller välja bort saker, alternativ eller händelser, men även för att kolla dess validitet. Det kan exempelvis röra sig om att kolla huruvida ett påstående är sant eller falskt och som följd av det göra något som specificerats i koden. Det skulle kunna vara att starta ett spel när du klickar på ett specifikt element på hemsidan eller i programmet om vi tänker i andra språk.

Finns kodexempel nedan:

x = 10

if (x <= 11) {

x += 1;

Console.Writeln(x);

}

For-sats
